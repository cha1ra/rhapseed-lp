# Rhapseed LP

Rhapseed landing page based on Svelte & Tailwind CSS
[https://rhapseed.com/](https://rhapseed.com/)
  
## Get started

Install the dependencies...

```bash
git clone https://github.com/cha1ra/rhapseed-lp.git
cd svelte-app
npm install
```

...then start [Rollup](https://rollupjs.org):

```bash
npm run dev
```
